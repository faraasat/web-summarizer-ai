# Installation Guide

```shell
    python -m venv .venv

    # for linux/mac
    source .venv/bin/activate

    # For Windows
    .venv/Scripts/activate.bat # In CMD
    .venv/Scripts/Activate.ps1 # In Powershell

    pip install -r requirements.txt

    pip install -r requirements-dev.txt # only for dev
```
